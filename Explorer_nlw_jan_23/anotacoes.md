# 1. Objetos

atributos -> São as propriedades de um objeto
métodos -> São as funcionalidades de um objeto

```javascript
// criando um objeto
const celular = {
  cor: 'preto',
  ligar: function() {
    const mensagem = "Ligando"
    alert(mensagem)
  } //método
}

// usando um objeto
  celular.cor //preto
  celular.ligar() //executa a função, usa parentesis pq é função
```
